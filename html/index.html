Traduzione in italiano documentazione FreeBSD

Pagina di riferimento e coordinazione progetto: https://wiki.freebsd.org/ItalianDocumentationProject.

Lo strumento di riferimento per la traduzione è zanata.

Il nome del progetto sul sito di Zanata è "FreeBSD-it_IT".

Chi volesse lavorare offline può usare il file ".po" dispnibile su github (link), clonando il repositorio e inviando le modifiche come pull request.

La traduzione avviene usando i file ".po" di gettext, che dividono il testo da tradurre in pezzi. È possibile lavorare offline con i port editors/poedit o devel/gtranslator, ma salvo non abbiate confidenza con git github, gettext e il software che usate consiglio di usare l'interfaccia web di Zanata.

Qui trovate la versione renderizzata della traduzione fatta fino adesso. (attualmente ho dei problemi di encoding nel generare l'html, sto studiando una soluzione)

IMPORTANTE: leggete le linee guida per la traduzione.
